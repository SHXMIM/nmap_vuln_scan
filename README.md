# nmap_vuln_scan

### Installation

```
git clone https://github.com/SHXMIM/nmap_vuln_scan.git
```
```
cd nmap_vuln_scan
```
```
chmod 755 nmap_vuln_scan.sh
```
```
./nmap_vuln_scan.sh
``` 
### Example

```
./nmap_vuln_scan.sh 
```
